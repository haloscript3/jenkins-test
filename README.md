# JenkinsTest Maven Project

This is a test Maven project created as an example repo for  **Jenkins CI/CD pipelines**.  
The project contains sample Java files, database connection code, and basic Maven configuration to test builds and automated processes.

## Project Structure

```
JenkinsTest/
│
├─ pom.xml                 # Maven project configuration
├─ src/
│   ├─ main/
│   │   └─ java/
│   │       └─ com/neotech/sisdb/
│   │           └─ ConnectMSSQL.java   # Sample database connection class
│   └─ test/
│       └─ java/
│           └─ com/neotech/sisdb/
│               └─ (unit tests if any)
```
